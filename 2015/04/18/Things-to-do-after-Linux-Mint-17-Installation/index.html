Things to do after Linux Mint 17 Installation

Apr 18, 2015

In following paragraphs, Storage(NTFS format) refers to a drive that is being shared between Windows and Mint.

  1. Change keyboard layout to use Num Key as in windows(Miscellenous->Windows)

  2. Delete Downloads, Music, Video folders, etc and Drag and Drop same Folders from Storage Drive to the side pane.

  3. If firefox doesn't support Google, add an Addon. Google Default4. Fix Boot. 
    sudo update
    . Install Boot-Repair - Community Help Wiki
  4. Install Chrome. Download chrome from website, 
    sudo dpkg -i google-chrome-stable_current_i386.deb
  5. Set Download locations of chrome and firefox to Storage->Downloads->Chrome
  6. Install Geany IDE. 
    sudo apt-get install geany
  7. Install libraries for C compilation. 
    sudo apt-get install build-essential
  8. If brightness buttons don't work. Fix Brightness Control Not Working for Ubuntu 14.04 & Linux Mint 17
  9. Install atom text editor. Install Atom Text Editor In Ubuntu 14.04 & Linux Mint 17
  10. Install Oracle Java. How to Completely Install Oracle JDK on Linux Mint 17. 
    wget –no-check-certificate –no-cookies –header "Cookie: oraclelicense=accept-securebackup-cookie" <Download URL>
