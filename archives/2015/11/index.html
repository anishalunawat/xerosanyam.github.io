Java on Linux

Nov 04, 2015

  1. Set PATH: used to run javac & other bin applications from anywhere. Otherwise you need to give the complete path. (Was set automatically, somehow)

  2. Set CLASSPATH: used to look for jdk tools & user classes. You won't be able to use any class from current directory if you do no set it.In Java it is important to know from where you are compiling your program.
